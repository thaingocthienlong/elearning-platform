<objective>
Create the shared server-only media entitlement contract that all protected media surfaces can consume.
</objective>

<must_haves>
- One helper under `src/lib/media-entitlement.ts` is the source of truth for media access.
- Helper enforces authenticated user, user exists, video exists, video not deleted, video published, open-course access, enrollment, direct video access, direct-access date windows, and view limits.
- Helper returns typed result objects and denial codes rather than throwing for normal authorization failures.
- Helper exposes sanitized HTTP mapping for route handlers.
- Tests cover allow/deny paths before routes are refactored.
</must_haves>

<tasks>
